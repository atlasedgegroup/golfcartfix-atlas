Yamaha Golf Cart Controller Problems

Drive (DC) and Drive2 (AC) controller diagnosis. Confirm battery, solenoid, and throttle sensor before replacing the controller.

Fast Answer: DC vs AC System

Drive (2007–2016, DC motor): Test output voltage at motor terminals. No output with solenoid closed and battery voltage present = controller failure. Drive2 (2017+, AC motor): Check dash error indicator blink codes first. AC system faults are more specific and diagnosed differently than DC.

Drive DC

Probe B+ and motor output terminals during pedal press. No ramp-up in voltage = controller issue.

Drive2 AC

Note dash LED blink pattern. AC controller and motor are matched; don't replace one without diagnosing the other.

G-Series Electric

Simpler mechanical speed controller on older models. Often can be rebuilt vs replaced.

Eliminate These First

Controller replacement is expensive ($250–$700). Before ordering one, confirm:

  • Battery pack voltage is in range (50–52V for Drive/Drive2)
  • Solenoid is confirmed closed during pedal press (continuity test on large posts)
  • Throttle potentiometer or Hall sensor output is smooth (no gaps in resistance curve)
  • Speed sensor is reading correctly (not sending bad feedback that causes controller to limit output)

Drive (DC) Controller Test

Set multimeter to DC voltage. With key on, solenoid confirmed closed, probe the B+ (battery positive) terminal on the controller — should read pack voltage. Now probe the M (motor output) terminal during a slow pedal press. Voltage should ramp smoothly from 0 to near pack voltage. Zero output from M with correct input at B+ = controller failed.

Drive2 (AC) Controller

The Drive2 uses an AC induction motor with regenerative braking, controlled by an AC motor controller. This system is significantly more integrated than the DC system. The controller and motor are factory-matched. If replacing one, the other should be tested. AC fault diagnosis without specialized equipment is limited — if the cart is showing a fault code and the above eliminations are clear, take it to a Yamaha dealer or a shop with AC diagnostic equipment before buying parts.
